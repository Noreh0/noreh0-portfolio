# Noreh0 Portfolio

Portfólio pessoal focado em Engenharia de Software, DevOps e Cloud, com interface moderna e animações interativas.

## Tecnologias utilizadas

### Base do projeto
- HTML5
- CSS3
- JavaScript (ES6+)

### Bibliotecas e recursos externos
- Lenis (smooth scroll)
- GSAP (animações)
- GSAP ScrollTrigger (animações por rolagem)
- SplitType (quebra de texto para efeitos de reveal)
- anime.js (animações de cursor e efeitos visuais)
- Phosphor Icons (ícones)
- Google Fonts
	- Shippori Mincho
	- Zen Kaku Gothic New
	- JetBrains Mono
	- Google Sans Flex

## Recursos implementados no portfólio
- Rolagem suave com navegação por âncoras
- Cursor customizado com efeitos de interação
- Header dinâmico com comportamento no scroll
- Hero com animação em canvas
- Animações de entrada e transição com GSAP
- Relógios em tempo real (fusos de São Paulo e Nova York)
- Cards de projetos com efeito tilt 3D
- Sessão de clientes com animação stagger

## Como executar localmente
1. Clone este repositório.
2. Abra a pasta do projeto em um editor.
3. Execute o arquivo `index.html` no navegador.
